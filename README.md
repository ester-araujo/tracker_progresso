# ⚡ Study Tracker — Cyberpunk Edition

Um gerenciador de estudos e projetos de alta performance, focado em acompanhamento modular de progresso, desenvolvido com **Python**, **Streamlit** e **SQLite**.

---

## 🎨 Layout e Identidade Visual

O projeto conta com uma interface personalizada usando **CSS Injection** para criar um tema moderno com estética Cyberpunk/Neon.

* **Background:** `#0d0b72` (Roxo Profundo / Dark)
* **Sidebar:** `#F4F1EA` (Champanhe Claro)
* **Destaques & Títulos:** `#FF4191` (Neon Pink) e `#A3FFD6` (Mint Green)
* **Acentos:** `#A370F7` (Electric Lilac)

---

## 🚀 Funcionalidades Atuais

- [x] **Arquitetura SPA:** Navegação fluida entre abas via barra lateral.
- [x] **Estilização Customizada:** Sobrescrita de componentes nativos do Streamlit com CSS próprio.
- [x] **Persistência de Dados:** Integração completa com banco de dados SQLite local.
- [x] **Cadastro de Projetos:** Formulário otimizado (`st.form`) para adicionar cursos, certificações e módulos.
- [x] **Dashboard de Progresso:** Visualização de projetos com métricas e barras de status.

---

## 🛠️ Tecnologias Utilizadas

- **Python 3.x**
- **Streamlit** (Interface Web)
- **SQLite3** (Banco de Dados Relacional)
- **HTML5 & CSS3** (Customização de UI/UX)
